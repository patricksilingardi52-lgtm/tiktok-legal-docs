YT Agent Demo - End-to-End Flow

This demo simulates the AI video automation workflow integrating with TikTok Content Posting API.

Step 1: YouTube Search

Search YouTube for videos using the Data API.

Step 2: Video Processing

Retrieve video from database, analyze with AI, generate short using Klap API.

Step 3: Upload to TikTok

Upload generated short to TikTok using Content Posting API (scopes: video.publish, user.info.basic).

Step 4: Upload to YouTube

Upload short to YouTube using Upload API.
